# PortManager - 桌面端口管理工具

![Language](https://img.shields.io/badge/Language-Go-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

现代化的 Windows 端口检测和管理工具，采用 G-Helper 风格的卡片式桌面窗口设计。

本项目已按开源项目整理，采用 MIT License 发布，见 [LICENSE](LICENSE)。

## ✨ 功能特性

- **🔍 端口扫描** - 支持快速扫描（常用/高风险端口）和全面扫描
- **📊 实时监控** - 显示关联的进程信息 (PID、进程名、路径)
- **⚙️ 一键管理** - 轻松关闭异常或不需要的端口
- **🚀 开机自启** - 支持开机自动启动
- **🎨 现代UI** - 受 G-Helper 启发的卡片式布局
- **💻 桌面界面** - 原生窗口式体验
- **⚡ 极速启动** - 轻量级设计，快速响应

##  快速开始

## 📋 使用说明

### 界面布局

PortManager 采用现代的卡片式布局，分为三个主要功能区：

#### 1. 🔍 端口扫描卡片
- **快速扫描**：优先扫描常用和高风险端口
- **全面扫描**：扫描系统当前全部监听端口
- 显示扫描状态和结果反馈
- 扫描完成后自动刷新结果

#### 2. 📊 扫描结果卡片
- 显示发现的开放端口总数
- 列出每个端口的详细信息：
  - 端口号
  - 关联进程名
  - 进程ID (PID)
- 为每个端口提供快速**关闭**按钮

#### 3. ⚙️ 设置卡片
- **开机自启动**：勾选启用/禁用程序开机自动启动
- **关于**：查看应用版本和功能描述

### 常见操作

#### 查看所有开放端口
1. 点击"快速扫描"或"全面扫描"按钮
2. 等待扫描完成
3. 在结果上查看所有端口和关联进程

#### 关闭特定端口
1. 快速扫描或全面扫描找到目标端口
2. 点击该端口右边的"关闭"按钮
3. 确认关闭操作
4. 端口被立即关闭

#### 启用开机自启动
1. 打开设置卡片
2. 勾选"开机自启动"复选框
3. 下次重启时 PortManager 会自动启动

## 💡 使用建议

### 第一次使用 (3 分钟)
1. 以管理员身份运行 `run.bat`
2. 程序自动打开，看到卡片式界面
3. 点击"快速扫描"查看高风险端口，或点击"全面扫描"查看全部监听端口
4. 尝试关闭一个不需要的端口

### 日常使用
- **定期扫描**: 每周扫描一次，检查是否有未知端口
- **设置开机自启**: 勾选"开机自启动"，让程序自动保护你的系统
- **快速关闭**: 一键关闭异常或未知的端口

### 注意事项

⚠️ **警告**：
- 请不要随意关闭系统关键端口（如 135, 445 等），否则可能导致系统功能异常
- 某些操作需要 **管理员权限**
- 关闭端口会终止相关进程，某些应用可能无法继续运行

## ❓ 常见问题

### Q: 为什么显示"需要管理员权限"？
**A**: 关闭端口和查看某些进程信息需要管理员权限。请右键选择"以管理员身份运行"，或使用 `run.bat` 脚本。

### Q: 为什么有些端口无法关闭？
**A**: 可能原因：
- 该端口由系统进程占用（如 svchost 等）
- 权限不足
- 进程已经退出

### Q: 如何卸载？
**A**: 直接删除 `PortManager.exe` 和 `run.bat` 即可。配置文件保存在 `%APPDATA%\PortManager\`。

### Q: 支持其他操作系统吗？
**A**: 目前仅支持 Windows 10 及以上版本。跨平台支持正在规划中。

### Q: 能监控流量吗？
**A**: 当前版本专注于端口检测和管理。流量监控功能正在开发中。
