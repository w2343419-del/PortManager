# PortManager

Windows 桌面端口管理工具（Go + Walk），聚焦三个事情：
- 快速发现本机监听端口
- 一键关闭选中端口
- 对端口给出右键分析与关闭建议

## 特性

- 双模式扫描：快速扫描 / 全面扫描
- 端口结果列表：端口、进程名、PID
- 右键端口分析：
  - 当前端口信息
  - 常见用途说明
  - 活跃连接数（作为流量指标）
  - 处理建议：建议关闭 / 建议开启 / 无需操作
- 一键关闭端口（会结束占用该端口的进程）
- 开机自启动开关
- Windows GUI 子系统 + 隐藏命令窗口

## 系统要求

- Windows 10/11
- 建议管理员权限运行（关闭端口、读取部分进程信息需要）

## 快速开始

### 1) 直接运行（推荐）

双击以下文件：
- run.bat

`run.bat` 会转调 `run.vbs`，以管理员权限启动 `PortManager.exe`。

### 2) 从源码构建

```powershell
go mod tidy
go build -ldflags "-H=windowsgui" -o PortManager.exe .
```

或双击：
- build_gui.bat

## 使用说明

### 扫描端口

1. 点击 `快速扫描` 或 `全面扫描`
2. 等待结果刷新
3. 在“扫描结果”里查看端口条目

### 关闭端口

1. 选中列表中的端口
2. 点击 `关闭端口`

### 右键端口分析（新增）

1. 在结果列表中选中端口
2. 右键 -> `查看端口信息与建议`
3. 弹窗会展示：
- 端口、协议、进程、PID
- 常见用途
- 活跃连接数与流量等级
- 建议与依据

> 建议逻辑会结合端口类型、进程属性与当前连接情况动态判断。

## 常见问题

### 1. 程序启动后没反应 / 一闪而过

优先使用 `run.bat` 启动，并确认 `PortManager.exe` 已构建成功。

### 2. 扫描或关闭失败

通常是权限不足。请以管理员身份运行。

### 3. 为什么关闭端口会影响应用

关闭端口本质是结束占用该端口的进程，请先确认该进程用途。

### 4. 最小化恢复后窗体跑位

当前版本已在恢复时自动修正尺寸与右下角位置；若仍异常，反馈显示器缩放比例与多屏布局。
