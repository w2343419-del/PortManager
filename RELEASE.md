# PortManager Release Notes

## v1.1.0 (2026-04-16)

### 概述

v1.1.0 聚焦于稳定性、交互一致性和可运维性，已达到日常使用和发布要求。

### 新增

- 系统托盘支持：关闭窗口后最小化到托盘。
- 单实例模式：重复启动时只激活已有窗口，不再创建第二个窗口。
- 端口洞察：右键查看用途、活跃连接、风险等级和建议动作。

### 优化

- 扫描流程优化：批量获取进程信息，减少命令调用开销。
- 扫描模式优化：快速扫描改为速度优先路径，和全面扫描形成明显时间差。
- 结果展示优化：端口条目增加协议显示（TCP/UDP）。
- 图标与样式优化：统一应用图标，启用现代控件样式。

### 修复

- 修复重复端口显示问题（同端口多协议去重）。
- 修复窗口最小化恢复后尺寸和位置漂移问题。
- 修复中文环境下 netstat 输出解析兼容性问题。

### 兼容性

- 系统：Windows 10/11
- 权限：建议管理员权限运行
- 构建环境：Go 1.25.5

### 已知限制

- 仅支持 Windows。
- 关闭端口的本质是结束占用该端口的进程，请先确认业务影响。
